# 📺 个人高清 IPTV 直播源分享

欢迎来到我的个人直播源托管仓库！这里主要整理并分享我自己最近找到的优质直播源，支持多种播放器在线订阅，内容涵盖新闻、电影、4K高清、港澳台及东南亚本地频道等。

---

## 🚀 快速订阅地址

你可以直接复制下方的 Raw 原始链接，填入支持 M3U 的播放器（如 PotPlayer, VLC, TiviMate, IPTV Pro 等）中直接收看：

👉 `https://raw.githubusercontent.com/你的GitHub用户名/你的仓库名/main/live.m3u`

> 💡 **小贴士**：请记得将链接中的 `你的GitHub用户名` 和 `你的仓库名` 替换为你自己在 GitHub 上实际创建的名称。
> 
> *(如果在中国大陆遇到 GitHub 链接加载缓慢或无法连接，可以使用 jsDelivr CDN 加速链接：`https://cdn.jsdelivr.net/gh/你的GitHub用户名/你的仓库名/live.m3u`)*
